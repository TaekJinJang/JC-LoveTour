## Teck Stack

- FrontEnd

  - React (SSR)
  - React Hooks
  - Redux / Redux Saga
  - Ant Design
  - Styled Components

- BackEnd

  - Node.js
  - Express
  - MySQL (Sequelize)
  - Bcrypt

- production
  - AWS
  - EC2 / S3
  - Route 53
